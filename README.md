# Accounts
Projeto de uma api para cadastro e login de conta.

## Documentação
Para melhor organização, a documentação foi disponibilizada na wiki do repositório, que pode ser acessada clicando [aqui](https://github.com/julioschuambach/accounts-api/wiki).

## Funcionalidades
**Usuário comum**
- Criação de conta, informando: nome de usuário, e-mail, e senha.
- Login de conta, informando: e-mail e senha.

**Administrador**
- Listagem de todas as contas registradas.
